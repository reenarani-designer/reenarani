Solar Energy App

This application enables users to manage their solar power and main electricity. Designed for solar panel power management, energy consumption statistics, and real-time data visualisations.

The problem

No control on solar system: The issue was discovered when some individuals were turning their solar system on and off manually and they were not able to control it if they are not at their homes.

No previous record: They don't have any record of their saved and consumed electricity.

The Solution

This application is designed for energy management to give users complete control over their solar energy and electricity. Key features include real-time data visualization, power control, detailed insights into solar energy production, and energy consumption data. Users can easily switch their home electricity from solar to the main source and vice versa, even when they are not at home.

Process

  • User Needs: Identifying users needs through interviews and user surveys.
  • Analysis: Created user personas and user journey map to understand the user's pain points
  • Design Ideas: I did brainstorming sessions to get the best possible solution for this app.
  • Prototype: Developed low-fidelity prototyping to make a visual understanding of the app and then converted it into high-fi prototyping
  • Testing: At the end of this process I iterated my designs based on user testing to create a more accessible and user centered product.

Results

I was able to deliver a project according to the users need within a timeframe. All the users found the app easy to use with a good user experince.

But during the user testing I also discussed about the more features in the app, so I got some suggestions like home appliances control and some energy savings content in the app. If I got an opportunity to work on this app so I would like to add those features in this application.

This project combined my interest in sustainable energy with my dedication to designing meaningful user experiences. It was incredibly rewarding to create a tool that not only solves a real-world problem but also encourages more people to adopt clean energy.
